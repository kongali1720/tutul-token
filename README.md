# TUTUL Token (TTL)

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)

**TUTUL Token (TTL)** adalah ERC20-compliant token di jaringan Ethereum, dibuat untuk **utility dan proyek sosial**.  
Token ini memiliki supply awal **100,000,000 TTL**, mintable oleh owner, dan burnable.
